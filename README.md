# HEALTH-AI: ML Learning Tool for Healthcare
This project is developed as part of the Software Quality Assurance course.

## 🛠 Tech Stack
- **Frontend:** React 18+ (Vite)
- **Backend:** FastAPI (Python)
- **ML Engine:** Scikit-learn
- **Design:** Figma

## 🚀 Key Features
- **7-Step Journey:** Clinical Context to Ethics & Bias.
- **Model Tuning:** Real-time visualization of 6 AI algorithms.
- **Privacy First:** Entirely browser-based data processing.
